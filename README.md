CODINGSAMURAI - Data Analysis Projects

This repository contains two Exploratory Data Analysis (EDA) projects built using Python.

---

📊 Project 1: Titanic Survival Analysis

Objective

To analyze passenger data and understand factors affecting survival during the Titanic disaster.

Key Insights

- Women and children had higher survival rates.
- First-class passengers had better survival chances.
- Fare and passenger class are strongly related to survival.
- Missing values in age and deck limited deeper analysis.

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly

---

🛒 Project 2: Superstore Sales Analysis

Objective

To analyze sales performance, customer behavior, and profitability patterns.

Key Insights

- Business shows stable but non-differentiated performance.
- Customer ratings are not strongly linked to spending.
- Revenue is evenly distributed across segments.
- Gross margin is uniform across products.
- Growth is driven by volume, not strong segment advantage.

Limitations

- Only 3 months of data available.
- Limited number of business variables.
- Only 3 branches included in dataset.

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly
